# OfferPilot

Your job search command center.

OfferPilot is a full-stack job-search CRM that helps users track applications, manage follow-ups, monitor progress, and turn their job search into a clear, organized workflow.

## Tech Stack

### Backend

- Django
- Django REST Framework
- PostgreSQL

### Frontend

- Next.js
- TypeScript
- TanStack Query
- Tailwind CSS
